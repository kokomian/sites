Placement

    Goals of Placement :

    • In Placement stage we will place standard cells legally in the standard cell rows created in floor plan stage.
    • Placement should be done in such a way that it leave enough space for adding new cells or upsizing existing cells in future and routing nets while satisfying all the timing and routing constraints.

    Steps followed in the placement :

    • S1 : Create library,Read the Power Plan database, Perform necessary checks and Clean Violations if any found.
    • S2 : Fix macros and ports.
    • S3 : Assign Metal layer Routing directions.
    • S4 : Grep the cells that are not used during placement.
    • S5 : Add Tap cells.
    • S6 : Set Design Rule Constraints.
    • S7 : Set required Application options
    • S8 : Perform placement,legalize the placement and optimization.
    • S9 : Perform appropriate checks and generate appropriate reports.
    • S10 : Try to reduce the violations by choosing suitable techniques.
  • Design Import and Pre-Checks :-
    • Create library
    • Read the power plan database
    • Perform required Pre-checks and generate appropriate reports
    • Clean the violations if any found
    • Create timing environment

CTS

    Goals of CTS :

    • Every flop in the design should get clock.
    • Clock Tree should be build with minimum insertion delay and minimum local skew.
    • Clock Tree can be build by using Non-Default rules to reduce crosstalk issues.

    Steps followed during Clock Tree Synthesis :

    • S1 : Create library,load placement database,perform necessary checks and clean violations if any found.
    • S2 : Grep required cells and building clock tree assign spacing b/w clock cells.
    • S3 : Assign Metal layer Routing directions.
    • S4 : Create and set Metal layer routing rules.
    • S5 : Set Design Rule Constraints and target skew.
    • S6 : Set target skew.
    • S7 : Set application options.
    • S8 : Build clock tree,route clock tree and perform clock tree optimization.
    • S9 : Perform necessary checks and generate appropriate reports.
    • S10 : Try to reduce violations by suitable optimization techniques.
  • Design Import:-
    • Create library
    • Read the placement database
    • Perform required checks and generate appropriate reports
    • Clean the violations if any found
    • Create timing environment

  • CT references
